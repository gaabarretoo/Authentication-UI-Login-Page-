## • Sobre o projeto

Este projeto consiste no desenvolvimento de uma **tela de login moderna e responsiva**, com foco em **experiência do usuário (UX)**, design limpo e boas práticas de desenvolvimento front-end.

A interface apresenta um visual elegante com efeito glassmorphism e paleta em tons de roxo, simulando um cenário real de autenticação em sistemas web.

---

## •	Tecnologias utilizadas

* HTML5
* CSS3 

---

## •	Funcionalidades

*  Interface de login moderna e intuitiva
*  Campos com ícones e feedback visual
*  Layout responsivo (desktop e mobile)
*  Opção "lembrar senha"
*  Recuperação de senha
*  Estrutura organizada e reutilizável
